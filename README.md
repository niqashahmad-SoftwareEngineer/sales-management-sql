# Sales Management System — SQL Project

## Overview
A complete SQL database project simulating
a real business sales management system.

## Database Schema
- Customers — stores customer information
- Products — stores product catalog
- Orders — stores order headers
- Order Items — stores order line items

## Business Queries Included
1. Total Revenue Analysis
2. Revenue Per Customer
3. Best Selling Products
4. Revenue By City
5. Monthly Revenue Trends
6. Top Customer Per City
7. Products Never Ordered
8. Pending Orders Report
9. Average Order Value
10. Category Performance

## Skills Demonstrated
- Table Design with Foreign Keys
- Complex JOINs across 3 tables
- Aggregate Functions
- GROUP BY and HAVING
- Subqueries
- Window Functions
